# AI Resume Analyser 🎯

Built this to practice PDF parsing and Gemini API integration.
Upload your resume, paste a job description, optionally add 
the company name — get an honest match score and feedback.

## What it does
- Upload resume as PDF
- Paste job description
- Add company name for targeted feedback
- Get match score with colour coded result
- See full analysis and suggested Git commits

## What I learned
- Extracting text from PDFs using pypdf
- Connecting Gemini API safely with dotenv
- Handling API errors and retry logic
- Building Streamlit UI with columns and tabs

## Tech used
Python, Streamlit, Gemini API, pypdf, python-dotenv

##  How to use
pip install -r requirements.txt
streamlit run app.py
